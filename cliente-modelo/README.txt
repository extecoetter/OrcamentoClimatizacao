cliente-modelo (PWA)

Arquivos:
- index.html: o app
- config.js: dados do instalador (troque por cliente)
- logo.png: logo do cliente (substitua mantendo o nome logo.png)
- manifest.json / service-worker.js / icons/: aparência de app (instalável)

Como criar um novo cliente:
1) Copie a pasta inteira cliente-modelo para cliente-NOME/
2) Troque logo.png pelo logo do cliente (mantendo o nome)
3) Edite config.js com nome/whats/pix do cliente
4) (Opcional) Edite manifest.json: "name" e "short_name" para aparecer no celular
